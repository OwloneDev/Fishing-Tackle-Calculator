# Калькулятор сборки снастей для игры [*"Русская Рыбалка 4"*](https://rf4game.ru).
Удобный онлайн-калькулятор для сборки сбалансированной снасти. Калькулятор рассчитан на безопасную ловлю от фрикциона и практически гарантирует, что при правильной сборке и вываживании слишком сильной рыбы вместо поломки удилища/катушки оборвётся леска/поводок.

## Как пользоваться
1. Откройте [веб-страницу калькулятора](https://owlonedev.github.io/Fishing-Tackle-Calculator/).
2. Выберите тип снасти из выпадающего списка.
3. Выберите нужную снасть из следующего выпадающего списка.
4. Продлеайте шаги 1-2 для всех необходимых видов снастей (*Удилище*, *Катушка*, *Леска*).
5. Поставьте галочку *Учитывать*, если в сборке используется поводок, затем введите его прочность.
6. Проанализируйте информацию о сборке. Если в графе *Советы* есть красные или жёлтые предупреждения, попробуйте подобрать другие снасти в соответствии с советом.
7. Если в графе *Советы* появилась зелёная надпись *"Оптимальная сборка!"*, то все снасти подобраны правильно.
