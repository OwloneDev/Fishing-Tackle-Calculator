# Калькулятор сборки снастей для игры [*"Русская Рыбалка 4"*](https://rf4game.ru).
Удобный онлайн-калькулятор для сборки сбалансированной снасти. Калькулятор рассчитан на безопасную ловлю от фрикциона и практически гарантирует, что при правильной сборке и вываживании слишком сильной рыбы вместо поломки удилища/катушки оборвётся леска/поводок.

## Как пользоваться
1. Выберите тип снасти из выпадающего списка.
2. Выберите нужную снасть из следующего выпадающего списка.
3. Продлеайте шаги 1-2 для всех необходимых видов снастей (*Удилище*, *Катушка*, *Леска*).
4. Поставьте галочку *Учитывать*, если в сборке используется поводок, затем введите его прочность.
5. Проанализируйте информацию о сборке. Если в графе *Советы* есть красные или жёлтые предупреждения, попробуйте подобрать другие снасти в соответствии с советом.
6. Если в графе *Советы* появилась зелёная надпись *"Оптимальная сборка!"*, то все снасти подобраны правильно.
